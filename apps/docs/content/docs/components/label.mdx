---
title: Label
description: A label component for form fields with built-in accessibility features.
---

import { Checkbox, Input, Label } from "uilab-core"

The Label component is built on Radix UI and provides accessible labels for form controls.

## Examples

### Default Label

<ComponentPreview title="Default Label">
  <div className="flex flex-col gap-2">
    <Label htmlFor="email">Email</Label>
    <Input id="email" type="email" placeholder="Enter your email" />
  </div>
</ComponentPreview>

```tsx
import { Input, Label } from "uilab-core"

export function Example() {
  return (
    <div className="flex flex-col gap-2">
      <Label htmlFor="email">Email</Label>
      <Input id="email" type="email" placeholder="Enter your email" />
    </div>
  )
}
```


## Props

The Label component accepts all standard HTML label attributes including:

- `htmlFor`: `string` - The id of the form element this label is associated with
- `asChild`: `boolean` - When true, the component will render its child instead of a label element
