---
title: Button Group
description: A group of buttons arranged horizontally or vertically with seamless borders.
---

import {
  Button,
  ButtonGroup,
  ButtonGroupSeparator,
  ButtonGroupText,
} from "uilab-core"

The ButtonGroup component allows you to group related buttons together with seamless borders and separators.

## Examples

### Default Button Group

<ComponentPreview title="Default Button Group">
  <ButtonGroup>
    <Button>First</Button>
    <Button>Second</Button>
    <Button>Third</Button>
  </ButtonGroup>
</ComponentPreview>

```tsx
import { Button, ButtonGroup } from "uilab-core"

export function Example() {
  return (
    <ButtonGroup>
      <Button>First</Button>
      <Button>Second</Button>
      <Button>Third</Button>
    </ButtonGroup>
  )
}
```

## Components

- **ButtonGroup**: The container for grouped buttons
- **ButtonGroupText**: A text element that can be placed between buttons
- **ButtonGroupSeparator**: A visual separator between buttons

## Props

### ButtonGroup

- `orientation`: `"horizontal" | "vertical"` - The direction of button arrangement (default: `"horizontal"`)
