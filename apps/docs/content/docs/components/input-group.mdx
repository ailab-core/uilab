title: Input Group
description: Group inputs with addons and icons.

import { InputGroup, InputGroupAddon, Input } from "uilab-core"

The Input Group component combines inputs with prefixes, suffixes, and icons.

## Examples

### With Prefix

<ComponentPreview title="Input Group with Prefix">
  <InputGroup>
    <InputGroupAddon>https://</InputGroupAddon>
    <Input placeholder="example.com" />
  </InputGroup>
</ComponentPreview>

```tsx
import { InputGroup, InputGroupAddon, Input } from 'uilab-core'

export function Example() {
  return (
    <InputGroup>
      <InputGroupAddon>https://</InputGroupAddon>
      <Input placeholder="example.com" />
    </InputGroup>
  )
}
```

## Components

- **InputGroup**: The root input group container
- **InputGroupAddon**: Prefix or suffix text addon

## Usage

Input Group automatically handles styling and positioning of addons and icons relative to the input element.
