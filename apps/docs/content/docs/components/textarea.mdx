The Textarea component provides a multi-line text input with auto-resizing support.

### With Controlled Value

<ComponentPreview title="Controlled Textarea">
  <Textarea
    placeholder="Type something..."
    value="Controlled value"
    onChange={(e) => console.log(e.target.value)}
  />
</ComponentPreview>

```tsx
import { Textarea } from 'uilab-core'
import { useState } from 'react'

export function Example() {
  const [value, setValue] = useState('')

  return (
    <Textarea
      placeholder="Type something..."
      value={value}
      onChange={(e) => setValue(e.target.value)}
    />
  )
}
```

### With Character Count

<ComponentPreview title="Textarea with Character Count">
  <div className="flex flex-col gap-2">
    <Label htmlFor="feedback">Feedback</Label>
    <Textarea id="feedback" placeholder="Share your feedback..." maxLength={200} />
    <p className="text-sm text-muted-foreground text-right">
      0 / 200 characters
    </p>
  </div>

```tsx
import { Textarea, Label } from 'uilab-core'
import { useState } from 'react'

export function Example() {
  const [value, setValue] = useState('')
  const maxLength = 200

  return (
    <div className="flex flex-col gap-2">
      <Label htmlFor="feedback">Feedback</Label>
      <Textarea
        id="feedback"
        placeholder="Share your feedback..."
        value={value}
        onChange={(e) => setValue(e.target.value)}
        maxLength={maxLength}
      />
      <p className="text-sm text-muted-foreground text-right">
        {value.length} / {maxLength} characters
      </p>
    </div>
  )
}
```

## Props

The Textarea component accepts all standard HTML textarea attributes including:

- `placeholder`: `string` - Placeholder text
- `disabled`: `boolean` - When true, prevents interaction
- `required`: `boolean` - When true, indicates required field
- `maxLength`: `number` - Maximum character limit
- `rows`: `number` - Number of visible text lines
- `value`: `string` - The controlled value
- `defaultValue`: `string` - The default value (uncontrolled)
- `onChange`: `(event: ChangeEvent<HTMLTextAreaElement>) => void` - Event handler for value changes

## Features

The Textarea component includes automatic content-based resizing using `field-sizing: content`, allowing it to grow and shrink based on the content without requiring JavaScript.
