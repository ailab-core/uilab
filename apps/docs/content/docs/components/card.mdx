The Card component provides a flexible container for grouping related content.

## Examples

### Basic Card

<ComponentPreview title="Basic Card">
  <Card className="w-[350px]">
    <CardHeader>
      <CardTitle>Card Title</CardTitle>
      <CardDescription>Card description goes here.</CardDescription>
    </CardHeader>
    <CardContent>
      <p>This is the card content area where you can place any content.</p>

    <CardFooter>
      <Button variant="outline">Cancel</Button>

    </CardFooter>
  </Card>
</ComponentPreview>

```tsx
import {
  Button,
  Card,
  CardContent,
  CardDescription,
  CardFooter,
  CardHeader,
  CardTitle,
} from "uilab-core"

export function Example() {
  return (
    <Card className="w-[350px]">
      <CardHeader>
        <CardTitle>Card Title</CardTitle>
        <CardDescription>Card description goes here.</CardDescription>
      </CardHeader>
      <CardContent>
        <p>This is the card content area.</p>
      </CardContent>
      <CardFooter>
        <Button variant="outline">Cancel</Button>
        <Button>Submit</Button>
      </CardFooter>
    </Card>
  )
}
```

## Components

The Card system includes several sub-components:

- `Card` - The main container
- `CardHeader` - Header section (optional)
- `CardTitle` - Title text with proper styling
- `CardDescription` - Description text with muted styling
- `CardContent` - Main content area
- `CardFooter` - Footer section (optional)
