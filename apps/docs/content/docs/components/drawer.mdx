import { Drawer, DrawerTrigger, DrawerContent, DrawerHeader, DrawerFooter, DrawerTitle, DrawerDescription, Button } from "uilab-core"

The Drawer component provides a mobile-optimized bottom sheet with drag-to-close functionality.

      </DrawerContent>


## Components

- **Drawer**: The root drawer component
- **DrawerTrigger**: The button that opens the drawer
- **DrawerContent**: The drawer content container
- **DrawerHeader**: Container for title and description
- **DrawerFooter**: Container for action buttons
- **DrawerTitle**: The drawer title
- **DrawerDescription**: The drawer description
- **DrawerClose**: Close button component

## Props

### Drawer

- `open`: `boolean` - The controlled open state
- `onOpenChange`: `(open: boolean) => void` - Open state change handler
- `shouldScaleBackground`: `boolean` - Whether to scale the background
- `snapPoints`: `number[]` - Snap points for drawer height
- `direction`: `"top" | "right" | "bottom" | "left"` - Drawer direction

Visit [Vaul documentation](https://vaul.emilkowal.ski/) for complete API reference.
