title: Alert

import { Alert, AlertTitle, AlertDescription } from 'uilab-core'

The Alert component is used to display important messages to users in a visually prominent way.

## Examples

### Default Alert

<ComponentPreview title="Default Alert">
  <Alert>
    <AlertTitle>Heads up!</AlertTitle>
    <AlertDescription>
      You can add components to your app using the CLI.
    </AlertDescription>
  </Alert>
</ComponentPreview>

```tsx
import { Alert, AlertTitle, AlertDescription } from 'uilab-core'

export function Example() {
  return (
    <Alert>
      <AlertTitle>Heads up!</AlertTitle>
      <AlertDescription>
        You can add components to your app using the CLI.
      </AlertDescription>
    </Alert>
  )
}
```

## Components

- `Alert` - The root container
- `AlertTitle` - Title text with emphasized styling
- `AlertDescription` - Description text with muted styling

## Props

### Alert

- `variant`: `"default" | "destructive"` - Visual style of the alert
