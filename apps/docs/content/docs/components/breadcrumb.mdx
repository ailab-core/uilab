title: Breadcrumb

import { Breadcrumb, BreadcrumbList, BreadcrumbItem, BreadcrumbLink, BreadcrumbPage, BreadcrumbSeparator, BreadcrumbEllipsis } from 'uilab-core'

The Breadcrumb component provides navigation showing the user's location within a hierarchy.

## Examples


## Components

- `Breadcrumb` - The root navigation container
- `BreadcrumbList` - Ordered list wrapper
- `BreadcrumbItem` - Individual breadcrumb item
- `BreadcrumbLink` - Link to a breadcrumb location
- `BreadcrumbPage` - Current page (non-clickable)
- `BreadcrumbSeparator` - Visual separator between items (defaults to chevron)
- `BreadcrumbEllipsis` - Ellipsis for collapsed items
