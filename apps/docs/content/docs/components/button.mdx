The Button component is a versatile, accessible button built on Radix UI primitives.

## Examples

### Default

<ComponentPreview title="Default Button">

</ComponentPreview>

```tsx
import { Button } from 'uilab-core'

export function Example() {
  return <Button>Click me</Button>
}
```

## Props

The Button component accepts all standard HTML button attributes plus:

- `variant`: `"default" | "secondary" | "destructive" | "outline" | "ghost" | "link"`
- `size`: `"default" | "sm" | "lg" | "icon"`
- `asChild`: boolean - Use the Radix Slot for composing with other components
