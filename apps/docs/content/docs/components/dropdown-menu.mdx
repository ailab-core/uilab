The Dropdown Menu component displays a menu of actions or options.

## Examples

```tsx
import {
  Button,
  DropdownMenu,
  DropdownMenuContent,
  DropdownMenuItem,
  DropdownMenuLabel,
  DropdownMenuSeparator,
  DropdownMenuTrigger,
} from "uilab-core"

export function Example() {
  return (
    <DropdownMenu>
      <DropdownMenuTrigger asChild>
        <Button variant="outline">Open Menu</Button>
      </DropdownMenuTrigger>
      <DropdownMenuContent>
        <DropdownMenuLabel>My Account</DropdownMenuLabel>
        <DropdownMenuSeparator />
        <DropdownMenuItem>Profile</DropdownMenuItem>
        <DropdownMenuItem>Billing</DropdownMenuItem>

```

## Components

- **DropdownMenu**: The root menu component
- **DropdownMenuTrigger**: The button that opens the menu
- **DropdownMenuContent**: The menu content container
- **DropdownMenuItem**: An individual menu item
- **DropdownMenuLabel**: A label for a section of items
- **DropdownMenuSeparator**: Visual separator between items
- **DropdownMenuGroup**: Groups related items
- **DropdownMenuCheckboxItem**: A checkable menu item
- **DropdownMenuRadioGroup**: Container for radio items
- **DropdownMenuRadioItem**: A radio menu item
- **DropdownMenuSub**: A submenu container
- **DropdownMenuSubTrigger**: Trigger for a submenu
- **DropdownMenuSubContent**: Submenu content
- **DropdownMenuShortcut**: Displays keyboard shortcut

## Props

### DropdownMenu

- `open`: `boolean` - The controlled open state
- `defaultOpen`: `boolean` - The default open state
- `onOpenChange`: `(open: boolean) => void` - Open state change handler
- `modal`: `boolean` - Whether the menu is modal

### DropdownMenuItem

- `inset`: `boolean` - Whether to indent the item
- `variant`: `"default" | "destructive"` - The visual style variant
- `disabled`: `boolean` - When true, prevents interaction
