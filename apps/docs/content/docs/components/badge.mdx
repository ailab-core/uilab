title: Badge
description: Displays a badge or label with different visual styles.

import { Badge } from "uilab-core"

The Badge component displays small labeled UI elements for categorization or status.

## Examples

## Props

- `variant`: `"default" | "secondary" | "destructive" | "outline"` - Visual style
- `asChild`: boolean - Use the Slot component for custom rendering
