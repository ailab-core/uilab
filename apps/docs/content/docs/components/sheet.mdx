import { Sheet, SheetTrigger, SheetContent, SheetHeader, SheetFooter, SheetTitle, SheetDescription, Button } from "uilab-core"

The Sheet component displays content in a panel that slides in from the screen edge.

## Components

- **Sheet**: The root sheet component
- **SheetTrigger**: The button that opens the sheet
- **SheetContent**: The sheet content container
- **SheetHeader**: Container for title and description
- **SheetFooter**: Container for action buttons
- **SheetTitle**: The sheet title
- **SheetDescription**: The sheet description
- **SheetClose**: Close button component

## Props

### Sheet

- `open`: `boolean` - The controlled open state
- `defaultOpen`: `boolean` - The default open state
- `onOpenChange`: `(open: boolean) => void` - Open state change handler

### SheetContent

- `side`: `"top" | "right" | "bottom" | "left"` - The side from which the sheet appears (default: `"right"`)
