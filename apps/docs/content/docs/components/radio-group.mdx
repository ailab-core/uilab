---
title: Radio Group
description: A set of checkable buttons where only one can be checked at a time.
---

import { RadioGroup, RadioGroupItem, Label } from "uilab-core"

The Radio Group component allows users to select a single option from a set of choices.

## Components

- **RadioGroup**: The root container for radio items
- **RadioGroupItem**: An individual radio button option

## Props

### RadioGroup

- `value`: `string` - The controlled value
- `defaultValue`: `string` - The default value (uncontrolled)
- `onValueChange`: `(value: string) => void` - Event handler called when the value changes
- `disabled`: `boolean` - When true, prevents interaction with all options
- `required`: `boolean` - When true, indicates that the user must select an option
- `orientation`: `"horizontal" | "vertical"` - The orientation of the radio group
- `name`: `string` - The name for form submission

### RadioGroupItem

- `value`: `string` - The value for this radio option (required)
- `disabled`: `boolean` - When true, prevents interaction with this specific option
- `required`: `boolean` - When true, indicates this option must be selected
