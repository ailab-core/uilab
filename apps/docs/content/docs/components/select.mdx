import { Select, SelectTrigger, SelectValue, SelectContent, SelectItem, SelectGroup, SelectLabel, SelectSeparator } from "uilab-core"

The Select component provides a dropdown list for selecting values.

## Examples

### Default Select

<ComponentPreview title="Default Select">
  <Select>
    <SelectTrigger className="w-[180px]">
      <SelectValue placeholder="Select a fruit" />
    </SelectTrigger>
    <SelectContent>
      <SelectItem value="apple">Apple</SelectItem>
      <SelectItem value="banana">Banana</SelectItem>
      <SelectItem value="orange">Orange</SelectItem>
    </SelectContent>
  </Select>

```tsx
import { Select, SelectTrigger, SelectValue, SelectContent, SelectItem } from 'uilab-core'

```

## Components

- **Select**: The root select component
- **SelectTrigger**: The button that triggers the dropdown
- **SelectValue**: Displays the selected value
- **SelectContent**: The dropdown content container
- **SelectItem**: An individual selectable item
- **SelectGroup**: Groups related items
- **SelectLabel**: Labels a group of items
- **SelectSeparator**: Visual separator between items

## Props

### Select

- `value`: `string` - The controlled value
- `defaultValue`: `string` - The default value (uncontrolled)
- `onValueChange`: `(value: string) => void` - Value change handler
- `disabled`: `boolean` - When true, prevents interaction
- `required`: `boolean` - When true, indicates required field
- `name`: `string` - Name for form submission

### SelectTrigger

- `size`: `"sm" | "default"` - The size of the trigger button

### SelectItem

- `value`: `string` - The value for this item (required)
- `disabled`: `boolean` - When true, prevents selection
