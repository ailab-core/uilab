import {
  Card,
  CardContent,
  Carousel,
  CarouselContent,
  CarouselItem,
  CarouselNext,
  CarouselPrevious,
} from "uilab-core"

The Carousel component displays scrollable content with navigation controls.

## Examples


## Components

- **Carousel**: The root carousel container
- **CarouselContent**: Wrapper for carousel items
- **CarouselItem**: Individual carousel slide
- **CarouselPrevious**: Previous navigation button
- **CarouselNext**: Next navigation button

## Props

### Carousel

- `orientation`: `"horizontal" | "vertical"` - Carousel orientation
- `opts`: Embla Carousel options object
- `plugins`: Embla Carousel plugins array
- `setApi`: Callback to access the Embla API

Visit [Embla Carousel documentation](https://www.embla-carousel.com/) for advanced options.
