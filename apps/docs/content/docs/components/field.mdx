import { Field, FieldLabel, FieldDescription, FieldError, Input } from "uilab-core"

The Field component provides a complete form field with label, description, and error messaging.

## Examples

### Default Field

<ComponentPreview title="Default Field">
  <Field>
    <FieldLabel>Email</FieldLabel>
    <Input type="email" placeholder="Enter your email" />
    <FieldDescription>We'll never share your email.</FieldDescription>
  </Field>
</ComponentPreview>

```tsx
import { Field, FieldLabel, FieldDescription, Input } from 'uilab-core'

export function Example() {
  return (
    <Field>
      <FieldLabel>Email</FieldLabel>
      <Input type="email" placeholder="Enter your email" />
      <FieldDescription>We'll never share your email.</FieldDescription>
    </Field>
  )
}
```

## Components

- **Field**: The root field wrapper
- **FieldLabel**: Label for the field
- **FieldDescription**: Helper text for the field
- **FieldError**: Error message display

## Usage

The Field component automatically handles accessibility by linking labels, descriptions, and errors with the input using proper ARIA attributes.
