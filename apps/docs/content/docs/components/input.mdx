import { Input, Label } from "uilab-core"

The Input component provides a styled text input field with support for various HTML input types.

## Examples

<ComponentPreview title="Default Input">
  <div className="w-full max-w-sm space-y-2">
    <Label htmlFor="email">Email</Label>
    <Input id="email" type="email" placeholder="Enter your email" />
  </div>
</ComponentPreview>

```tsx
import { Input, Label } from "uilab-core"

  return (
    <div className="space-y-2">
      <Label htmlFor="email">Email</Label>
      <Input id="email" type="email" placeholder="Enter your email" />
```

## Props

The Input component accepts all standard HTML input attributes including:

- `type`: Standard HTML input types (`text`, `password`, `email`, `number`, `date`, `file`, etc.)
- `placeholder`: Placeholder text
- `disabled`: Boolean to disable the input
- `className`: Additional CSS classes for styling
