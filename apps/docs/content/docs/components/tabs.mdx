## Components

- **Tabs**: The root container that manages tab state
- **TabsList**: Container for tab triggers
- **TabsTrigger**: A button that activates its associated tab content
- **TabsContent**: The content panel for each tab

## Props

### Tabs

- `value`: `string` - The controlled active tab value
- `defaultValue`: `string` - The default active tab (uncontrolled)
- `onValueChange`: `(value: string) => void` - Event handler called when the active tab changes
- `orientation`: `"horizontal" | "vertical"` - The orientation of the tabs
- `dir`: `"ltr" | "rtl"` - The reading direction

### TabsTrigger

- `value`: `string` - The value associated with this tab (required)
- `disabled`: `boolean` - When true, prevents interaction
- `asChild`: `boolean` - When true, renders as its child component
