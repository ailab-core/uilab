The Kbd component displays keyboard keys or shortcuts.

```tsx
import { Kbd } from 'uilab-core'
```

## Components

- **Kbd**: Displays a single keyboard key
- **KbdGroup**: Groups multiple keys together

## Usage

The Kbd component is ideal for documentation, tooltips, and UI hints to show keyboard shortcuts to users.
