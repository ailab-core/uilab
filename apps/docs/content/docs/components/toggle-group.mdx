## Components

- **ToggleGroup**: The root toggle group container
- **ToggleGroupItem**: An individual toggle button

## Props

### ToggleGroup

- `type`: `"single" | "multiple"` - Selection type (required)
- `value`: `string` - The controlled value (for single type)
- `defaultValue`: `string` - The default value (uncontrolled)
- `onValueChange`: `(value: string) => void` - Value change handler (single type)
- `disabled`: `boolean` - When true, prevents interaction with all items
- `variant`: `"default" | "outline"` - The visual style variant
- `size`: `"default" | "sm" | "lg"` - The size of toggle items

### ToggleGroupItem

- `value`: `string` - The value for this item (required)
- `disabled`: `boolean` - When true, prevents interaction
