---
title: Avatar
description: An image element with a fallback for representing a user.
---

import { Avatar, AvatarFallback, AvatarImage } from "uilab-core"

The Avatar component displays a user's profile picture with automatic fallback support.

## Examples

### Default Avatar

<ComponentPreview title="Default Avatar">
  <Avatar>
    <AvatarImage src="https://github.com/shadcn.png" alt="User" />
    <AvatarFallback>CN</AvatarFallback>
  </Avatar>
</ComponentPreview>

```tsx
import { Avatar, AvatarFallback, AvatarImage } from "uilab-core"

export function Example() {
  return (
    <Avatar>
      <AvatarImage src="https://github.com/shadcn.png" alt="User" />
      <AvatarFallback>CN</AvatarFallback>
    </Avatar>
  )
}
```

## Components

- `Avatar` - The root container
- `AvatarImage` - The image element
- `AvatarFallback` - Fallback content shown when image fails to load

## Props

### AvatarImage

- `src`: string - Image source URL
- `alt`: string - Alternative text for accessibility
