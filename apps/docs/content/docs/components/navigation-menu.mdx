## Components

- **NavigationMenu**: The root navigation menu
- **NavigationMenuList**: Container for menu items
- **NavigationMenuItem**: Individual menu item
- **NavigationMenuTrigger**: Button that opens submenu
- **NavigationMenuContent**: Dropdown content container
- **NavigationMenuLink**: Navigation link

## Usage

Navigation Menu is ideal for site-wide navigation with multiple levels and rich content in dropdowns.
