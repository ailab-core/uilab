title: Collapsible

import { Collapsible, CollapsibleTrigger, CollapsibleContent, Button } from "uilab-core"

The Collapsible component allows you to show and hide content with smooth transitions.

## Examples

### Default Collapsible

<ComponentPreview title="Default Collapsible">
  <Collapsible>
    <CollapsibleTrigger asChild>
      <Button variant="outline">Toggle Content</Button>
    </CollapsibleTrigger>
    <CollapsibleContent className="mt-2 p-4 border rounded-md">
      <p>This content can be collapsed and expanded.</p>
    </CollapsibleContent>
  </Collapsible>
</ComponentPreview>

```tsx
import { Collapsible, CollapsibleTrigger, CollapsibleContent, Button } from 'uilab-core'

export function Example() {
  return (
    <Collapsible>
      <CollapsibleTrigger asChild>
        <Button variant="outline">Toggle Content</Button>
      </CollapsibleTrigger>
      <CollapsibleContent className="mt-2 p-4 border rounded-md">
        <p>This content can be collapsed and expanded.</p>
      </CollapsibleContent>
    </Collapsible>
  )
}
```

### Controlled Collapsible

<ComponentPreview title="Controlled Collapsible">
  <Collapsible defaultOpen>
    <CollapsibleTrigger asChild>
      <Button variant="outline">Toggle</Button>
    </CollapsibleTrigger>
    <CollapsibleContent className="mt-2 space-y-2">
      <div className="rounded-md border px-4 py-3 text-sm">
        Item 1
      </div>
      <div className="rounded-md border px-4 py-3 text-sm">
        Item 2
      </div>
      <div className="rounded-md border px-4 py-3 text-sm">
        Item 3
      </div>
    </CollapsibleContent>
  </Collapsible>
</ComponentPreview>

```tsx
import { Collapsible, CollapsibleTrigger, CollapsibleContent, Button } from 'uilab-core'
import { useState } from 'react'

export function Example() {
  const [isOpen, setIsOpen] = useState(false)

  return (
    <Collapsible open={isOpen} onOpenChange={setIsOpen}>
      <CollapsibleTrigger asChild>
        <Button variant="outline">
          {isOpen ? 'Hide' : 'Show'} Details
        </Button>
      </CollapsibleTrigger>
      <CollapsibleContent className="mt-2 space-y-2">
        <div className="rounded-md border px-4 py-3 text-sm">
          Collapsible content here
        </div>
      </CollapsibleContent>
    </Collapsible>
  )
}
```

## Components

- **Collapsible**: The root component that manages state
- **CollapsibleTrigger**: The button that toggles the collapsed state
- **CollapsibleContent**: The content that can be collapsed

## Props

### Collapsible

- `open`: `boolean` - The controlled open state
- `defaultOpen`: `boolean` - The default open state (uncontrolled)
- `onOpenChange`: `(open: boolean) => void` - Event handler called when the open state changes
- `disabled`: `boolean` - When true, prevents the user from toggling
