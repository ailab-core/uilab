title: Tooltip

import { Button, Tooltip, TooltipContent, TooltipTrigger } from "uilab-core"

The Tooltip component displays informative text when users hover over or focus on an element.

## Components

- **Tooltip**: The root component that manages tooltip state
- **TooltipTrigger**: The element that triggers the tooltip on hover/focus
- **TooltipContent**: The content displayed in the tooltip
- **TooltipProvider**: Optional provider for configuring delay duration across multiple tooltips

## Props

### TooltipContent

- `side`: `"top" | "right" | "bottom" | "left"` - The preferred side of the trigger to render against
- `sideOffset`: `number` - The distance in pixels from the trigger (default: 0)
- `align`: `"start" | "center" | "end"` - The preferred alignment against the trigger
- `alignOffset`: `number` - An offset in pixels from the alignment

### TooltipProvider

- `delayDuration`: `number` - The duration from when the mouse enters a trigger until the tooltip opens (default: 0)
- `skipDelayDuration`: `number` - How long a user must hover before the skip delay kicks in

## Usage

Tooltips should be used to provide supplementary information that helps users understand an interface element. The Tooltip component automatically includes an arrow pointer and adjusts its position to stay within the viewport.
