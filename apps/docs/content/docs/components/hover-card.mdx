import { HoverCard, HoverCardTrigger, HoverCardContent, Button, Avatar, AvatarImage, AvatarFallback } from "uilab-core"

The Hover Card component displays content when hovering over a trigger element.

```tsx
import {
  HoverCard,
  HoverCardTrigger,
  HoverCardContent,
  Button,
  Avatar,
  AvatarImage,
  AvatarFallback
} from 'uilab-core'

export function Example() {
  return (
    <HoverCard>
      <HoverCardTrigger asChild>
        <Button variant="link">@username</Button>
      </HoverCardTrigger>
      <HoverCardContent className="w-80">
        <div className="flex justify-between space-x-4">
          <Avatar>
            <AvatarImage src="https://github.com/shadcn.png" />
            <AvatarFallback>VC</AvatarFallback>
          </Avatar>
          <div className="space-y-1">
            <h4 className="text-sm font-semibold">@username</h4>
            <p className="text-sm text-muted-foreground">
              Software engineer and open source contributor.
            </p>
          </div>
        </div>
      </HoverCardContent>
    </HoverCard>
  )
}
```

## Components

- **HoverCard**: The root hover card component
- **HoverCardTrigger**: The element that triggers the hover card
- **HoverCardContent**: The content displayed on hover

## Props

### HoverCard

- `open`: `boolean` - The controlled open state
- `defaultOpen`: `boolean` - The default open state
- `onOpenChange`: `(open: boolean) => void` - Open state change handler
- `openDelay`: `number` - Delay before opening (default: 700ms)
- `closeDelay`: `number` - Delay before closing (default: 300ms)

### HoverCardContent

- `side`: `"top" | "right" | "bottom" | "left"` - Preferred side
- `sideOffset`: `number` - Distance from trigger (default: 4)
- `align`: `"start" | "center" | "end"` - Alignment
