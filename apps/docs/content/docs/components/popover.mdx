The Popover component displays content in a floating panel.

## Components

- **Popover**: The root popover component
- **PopoverTrigger**: The button that triggers the popover
- **PopoverContent**: The popover content container
- **PopoverAnchor**: Optional anchor element for positioning

## Props

### Popover

- `open`: `boolean` - The controlled open state
- `defaultOpen`: `boolean` - The default open state
- `onOpenChange`: `(open: boolean) => void` - Open state change handler
- `modal`: `boolean` - Whether clicking outside closes the popover

### PopoverContent

- `side`: `"top" | "right" | "bottom" | "left"` - Preferred side
- `sideOffset`: `number` - Distance from trigger (default: 4)
- `align`: `"start" | "center" | "end"` - Alignment
- `alignOffset`: `number` - Alignment offset
