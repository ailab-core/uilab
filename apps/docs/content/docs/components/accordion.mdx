import { Accordion, AccordionItem, AccordionTrigger, AccordionContent } from 'uilab-core'

The Accordion component provides a collapsible content area built on Radix UI primitives.

## Examples

### Default Accordion

<ComponentPreview title="Default Accordion">
  <Accordion type="single" collapsible className="w-full">
    <AccordionItem value="item-1">
      <AccordionTrigger>Is it accessible?</AccordionTrigger>
      <AccordionContent>
        Yes. It adheres to the WAI-ARIA design pattern.
      </AccordionContent>
    </AccordionItem>
    <AccordionItem value="item-2">
      <AccordionTrigger>Is it styled?</AccordionTrigger>

    <AccordionItem value="item-3">
      <AccordionTrigger>Is it animated?</AccordionTrigger>
      <AccordionContent>
        Yes. It includes smooth animations for opening and closing.
      </AccordionContent>

  </Accordion>
</ComponentPreview>

```tsx
import { Accordion, AccordionItem, AccordionTrigger, AccordionContent } from 'uilab-core'

export function Example() {
  return (
    <Accordion type="single" collapsible>
      <AccordionItem value="item-1">

        </AccordionContent>
      </AccordionItem>
    </Accordion>
  )
}
```

## Components

The Accordion system includes several sub-components:

- `Accordion` - The root container (supports `type="single"` or `type="multiple"`)
- `AccordionItem` - Individual accordion item wrapper
- `AccordionTrigger` - Clickable trigger button that toggles content
- `AccordionContent` - The collapsible content area

## Props

### Accordion

- `type`: `"single" | "multiple"` - Whether one or multiple items can be open
- `collapsible`: boolean - Whether items can be collapsed when using `type="single"`
