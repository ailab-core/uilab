import { ScrollArea } from "uilab-core"

The Scroll Area component provides a customizable scrollable container with styled scrollbars.

## Examples

### Default Scroll Area

<ComponentPreview title="Default Scroll Area">
  <ScrollArea className="h-[200px] w-[350px] rounded-md border p-4">
    <div className="space-y-4">
      {Array.from({ length: 20 }).map((_, i) => (
        <div key={i} className="text-sm">
          Item {i + 1}: This is some scrollable content
        </div>
      ))}
    </div>
  </ScrollArea>
</ComponentPreview>

```tsx
import { ScrollArea } from "uilab-core"

export function Example() {
  return (
    <ScrollArea className="h-[200px] w-[350px] rounded-md border p-4">
      <div className="space-y-4">
        {Array.from({ length: 20 }).map((_, i) => (
          <div key={i} className="text-sm">
            Item {i + 1}: This is some scrollable content
          </div>
        ))}
      </div>
    </ScrollArea>
  )
}
```

### Horizontal Scroll

<ComponentPreview title="Horizontal Scroll Area">

    </div>
  </ScrollArea>
</ComponentPreview>

```tsx
<ScrollArea className="w-96 whitespace-nowrap rounded-md border">
  <div className="flex w-max space-x-4 p-4">{/* Horizontal content */}</div>
</ScrollArea>
```

## Props

The Scroll Area component accepts all Radix UI Scroll Area props. The scrollbar automatically appears when content overflows.

## Usage

Use Scroll Area when you need custom-styled scrollbars that match your design system. It provides better control over scroll behavior and appearance than native scrollbars.
