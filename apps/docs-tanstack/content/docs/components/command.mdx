## Components

- **Command**: The root command container
- **CommandInput**: Search input field
- **CommandList**: Scrollable list of commands
- **CommandEmpty**: Displayed when no results found
- **CommandGroup**: Groups related commands
- **CommandItem**: An individual command item
- **CommandSeparator**: Visual separator
- **CommandShortcut**: Displays keyboard shortcut

## Props

### Command

All cmdk props are supported. Key props include:

- `value`: `string` - The controlled value
- `onValueChange`: `(value: string) => void` - Value change handler
- `filter`: Function to customize filtering
- `shouldFilter`: `boolean` - Whether to enable filtering

### CommandItem

- `value`: `string` - The value for filtering
- `onSelect`: `(value: string) => void` - Selection handler
- `disabled`: `boolean` - When true, prevents interaction

Visit [cmdk documentation](https://cmdk.paco.me/) for complete API reference.
