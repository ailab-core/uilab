## Components

Menubar uses similar components to Dropdown Menu:

- **Menubar**: The root menubar container
- **MenubarMenu**: Individual menu in the bar
- **MenubarTrigger**: Button that opens the menu
- **MenubarContent**: Menu content container
- **MenubarItem**: Individual menu item
- **MenubarCheckboxItem**: Checkable menu item
- **MenubarRadioGroup**: Radio group container
- **MenubarRadioItem**: Radio menu item
- **MenubarLabel**: Label for menu sections
- **MenubarSeparator**: Visual separator
- **MenubarShortcut**: Keyboard shortcut display

The API is similar to Dropdown Menu. See Dropdown Menu documentation for detailed props.
