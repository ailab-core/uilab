## Components

- **ResizablePanelGroup**: Container for resizable panels
- **ResizablePanel**: An individual resizable panel
- **ResizableHandle**: Draggable handle between panels

## Props

### ResizablePanelGroup

- `direction`: `"horizontal" | "vertical"` - Layout direction (required)
- `onLayout`: `(sizes: number[]) => void` - Called when layout changes

### ResizablePanel

- `defaultSize`: `number` - Initial size as percentage
- `minSize`: `number` - Minimum size as percentage
- `maxSize`: `number` - Maximum size as percentage
- `collapsible`: `boolean` - Whether panel can collapse

### ResizableHandle

- `withHandle`: `boolean` - Whether to show grip handle icon

Visit [react-resizable-panels documentation](https://github.com/bvaughn/react-resizable-panels) for complete API reference.
