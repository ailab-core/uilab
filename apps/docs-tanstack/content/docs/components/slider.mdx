title: Slider

import { Label, Slider } from "uilab-core"

The Slider component allows users to select a value or range of values.

## Examples

### Default Slider

<ComponentPreview title="Default Slider">
  <Slider defaultValue={[50]} max={100} step={1} />
</ComponentPreview>

```tsx
import { Slider } from "uilab-core"

export function Example() {
  return <Slider defaultValue={[50]} max={100} step={1} />
}
```

### With Label and Value Display

<ComponentPreview title="Slider with Label">
  <div className="space-y-4">
    <div className="flex items-center justify-between">
      <Label>Volume</Label>
      <span className="text-muted-foreground text-sm">50%</span>
    </div>
    <Slider defaultValue={[50]} max={100} step={1} />
  </div>
</ComponentPreview>

```tsx
import { useState } from "react"
import { Label, Slider } from "uilab-core"

export function Example() {
  const [value, setValue] = useState([50])

  return (
    <div className="space-y-4">
      <div className="flex items-center justify-between">
        <Label>Volume</Label>
        <span className="text-muted-foreground text-sm">{value[0]}%</span>
      </div>
      <Slider value={value} onValueChange={setValue} max={100} step={1} />
    </div>
  )
}
```

## Props

- `value`: `number[]` - The controlled value(s)
- `defaultValue`: `number[]` - The default value(s) (uncontrolled)
- `onValueChange`: `(value: number[]) => void` - Event handler called when the value changes
- `min`: `number` - The minimum value (default: 0)
- `max`: `number` - The maximum value (default: 100)
- `step`: `number` - The step increment (default: 1)
- `disabled`: `boolean` - When true, prevents interaction
- `orientation`: `"horizontal" | "vertical"` - The orientation of the slider
