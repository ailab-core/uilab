The Chart components provide a set of composable chart primitives built on top of Recharts.

## Installation

The chart components require Recharts to be installed:

```bash
npm install recharts
```

## Usage

The chart components follow Recharts API and provide styled wrappers. Import the specific chart components you need from Recharts and use the provided theme colors.

## Example

```tsx
import {
  CartesianGrid,
  Legend,
  Line,
  LineChart,
  ResponsiveContainer,
  Tooltip,
  XAxis,
  YAxis,
} from "recharts"

const data = [
  { name: "Jan", value: 400 },
  { name: "Feb", value: 300 },
  { name: "Mar", value: 600 },
  { name: "Apr", value: 800 },
]

export function Example() {
  return (
    <ResponsiveContainer width="100%" height={300}>
      <LineChart data={data}>
        <CartesianGrid strokeDasharray="3 3" />
        <XAxis dataKey="name" />
        <YAxis />
        <Tooltip />
        <Legend />
        <Line type="monotone" dataKey="value" stroke="hsl(var(--primary))" />
      </LineChart>
    </ResponsiveContainer>
  )
}
```

## Theme Integration

Use CSS variables for chart colors to maintain theme consistency:

- `hsl(var(--primary))` - Primary color
- `hsl(var(--secondary))` - Secondary color
- `hsl(var(--muted))` - Muted color
- `hsl(var(--accent))` - Accent color

Visit [Recharts documentation](https://recharts.org/) for complete API reference.
