
## Components

- **InputOTP**: The root OTP input component
- **InputOTPGroup**: Groups related input slots
- **InputOTPSlot**: An individual digit input slot
- **InputOTPSeparator**: Visual separator between groups

## Props

### InputOTP

- `maxLength`: `number` - Maximum number of characters (required)
- `value`: `string` - The controlled value
- `onChange`: `(value: string) => void` - Value change handler
- `disabled`: `boolean` - When true, prevents input
- `pattern`: `string` - Regex pattern to validate input

Visit [input-otp documentation](https://input-otp.rodz.dev/) for complete API reference.
