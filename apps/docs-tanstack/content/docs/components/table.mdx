## Components

- **Table**: The root table wrapper with responsive overflow handling
- **TableHeader**: Groups table header rows
- **TableBody**: Contains the main table content rows
- **TableFooter**: Groups table footer rows
- **TableRow**: A row in the table
- **TableHead**: A header cell
- **TableCell**: A data cell
- **TableCaption**: Provides a caption or description for the table

## Usage

The Table component automatically handles overflow with horizontal scrolling on smaller screens. Use semantic table structure for better accessibility.
