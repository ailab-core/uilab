Welcome to the uilab-core component library! This collection includes 50+ production-ready React components built with Radix UI primitives and styled with Tailwind CSS.

## Installation

To use these components in your project:

```bash
pnpm add uilab-core
```

## Component Categories

### Form Components

- Button, Input, Textarea, Checkbox, Radio Group, Switch, Select, etc.

### Layout Components

- Card, Separator, Accordion, Tabs, Sidebar, etc.

### Overlay Components

- Dialog, Sheet, Popover, Dropdown Menu, Tooltip, etc.

### Data Display

- Table, Badge, Avatar, Calendar, Chart, etc.

### Feedback

- Alert, Progress, Spinner, Toast (Sonner), etc.

## Usage

All components support Tailwind CSS styling and are fully typed with TypeScript:

```tsx
import { Button, Card } from "uilab-core"

export function Example() {
  return (
    <Card>
      <Button>Click me</Button>
    </Card>
  )
}
```

Browse the individual component pages to see examples and usage details.
