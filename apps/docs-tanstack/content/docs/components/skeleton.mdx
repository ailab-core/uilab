---
title: Skeleton
description: A placeholder component for loading states.
---

import { Skeleton } from "uilab-core"

The Skeleton component displays a placeholder while content is loading.

## Examples

### Default Skeleton

<ComponentPreview title="Default Skeleton">
  <Skeleton className="h-12 w-full" />
</ComponentPreview>

```tsx
import { Skeleton } from "uilab-core"

export function Example() {
  return <Skeleton className="h-12 w-full" />
}
```

## Usage

Use Skeleton components to indicate loading states and improve perceived performance. They should match the approximate size and shape of the content they're replacing.
