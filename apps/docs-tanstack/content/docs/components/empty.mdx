## Components

- **Empty**: The root empty state container
- **EmptyHeader**: Container for icon, title and description
- **EmptyMedia**: Container for icon or illustration
- **EmptyTitle**: The empty state title
- **EmptyDescription**: The empty state description
- **EmptyContent**: Container for actions or additional content

## Props

### EmptyMedia

- `variant`: `"default" | "icon"` - Display style for the media element
